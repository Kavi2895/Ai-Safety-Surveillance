# Ai-Safety-Surveillance
AI-Based Safety Surveillance System is an intelligent safety monitoring solution designed for construction sites using Artificial Intelligence and Computer Vision. The system uses YOLO and OpenCV to detect safety violations such as missing helmets, safety vest non-compliance, unsafe worker behavior, and restricted area entry in real time. It provides live camera monitoring, instant alert notifications, and a centralized dashboard for tracking incidents and improving workplace safety. This project helps reduce accidents, automate safety surveillance, and enable proactive safety management.
